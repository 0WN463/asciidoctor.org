Dividing lists

Included in:

- user-manual

If you have adjacent lists, they have the tendency to want to fuse together.
To force the lists apart, place a line comment between them, offset on either side by a blank line (i.e., an end of list marker).
Here's an example:

[source]
----
* Apples
* Oranges
* Bananas

//^

* Walnuts
* Almonds
* Cashews
----
