== NAME

asciidoctor - converts AsciiDoc source files

== SYNOPSIS

*asciidoctor* ['OPTION']... 'FILE'...

== OPTIONS

*-o, --out-file*=_OUT_FILE_::
  Write output to file _OUT_FILE_.

*-q, --quiet*::
  Silence warnings.

== EXIT STATUS

*0*::
  Success.

*1*::
  Failure.
